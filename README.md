# Submission 3 - Malware Opcode Extraction(Project 1)
This repository contains the Submission 3 deliverables for the malware opcode extraction task.


## Contents of the ZIP Archive

The submitted .zip file contains:</br>
**Raw_Extracted_Files/** – the generated .opcode files for each malware sample. Each file follows the naming convention from Submission 2 to maintain a one-to-one mapping between the malware executable and the extracted opcode sequence.</br>
<br>
**Script/** – automation and extraction scripts used in the opcode extraction process, including run_extract.sh, OpcodeExport.java, the Python script used to extract opcodes from Windows and ELF binaries (extract_opcode.py), and the Python script used to extract opcode sequences from Android SMALI files generated from APK decompilation (extract_smali_opcode.py).
